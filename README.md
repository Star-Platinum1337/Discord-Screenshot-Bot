# Discord-Screenshot-Bot
A Discord bot that takes screenshots of URLs, splits them into parts, and sends them to chat


----------------------------------------------------------------------------------------------
uses python 3

libraries:
- discord.py
- Playwright
- Pillow
Enter this to terminal for instal --  pip install discord pytest-playwright pillow

COMMANDS:

!shot "Your URL"

Example:
!shot hhtps://github.com/



P.S  IN THE END CHANGE  bot.run('YOUR_BOT_TOKEN') TO YOUR BOT-TOKEN FROM DISCORD DEVELOPER PORTAL
